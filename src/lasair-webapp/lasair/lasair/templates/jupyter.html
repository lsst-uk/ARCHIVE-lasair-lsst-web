Lasair and Jupyter

Below are some Jupyter notebooks that use Lasair API. Full information about the api can be found at https://lasair-iris.roe.ac.uk/api You will need the "requests" package to run these (pip install requests). There are a few notebooks listed below.

  • Fetching lightcurves and plotting them (HTML, ipynb)
  • Finding objects in a sky rectangle with specific detection times (HTML, ipynb)
  • Objects near a PanStarrs galaxy and recently active (HTML, ipynb)
  • Fetching objects that are in a watchlist (HTML, ipynb)
There is another set of Jupyter notebooks that run directly on the Lasair database: more information here. Please Contact us with any notebooks that you would like to share.
